# CAD .step file

- Use these files if you wish to modify something. 
- The component "Cowl_Raw" is the cowl without any hotend or extruder mounting holes, useful for copying and creating additional cowls.  
- Keep in mind that the fan duct must blow air to the correct area: the tip of the hotend nozzle should sit 2 mm beneath the lowest part of the cowl.
