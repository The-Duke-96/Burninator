# STL files

- All printed parts are already in the correct printing orientation.
- Needed printing support is built in.
- Cosmetics and their corresponding parts need to be printed together; just align them and decide if and how you want to color them.
- LED_Diffuser, it is recommended to print it in either white or clear filament. Of course, you can also leave out the logo LED and print it in any color you want.
